# Machine Learning Course - Week 1

This repository contains week 1 materials and exercises for the machine learning course.

## Contents

- **week1.ipynb** - Main Jupyter notebook with course materials and exercises
- **data/** - Dataset files (CSV format)
  - wage.csv
  - smarket.csv
  - nci60_data.csv
  - Advertising.csv
  - college.csv
  - auto.csv
  - boston.csv

## Topics Covered

- Data exploration and visualization
- Wage data analysis
- Regression vs Classification
- Clustering (PCA and K-Means)
- Advertising data analysis
- Training vs Testing data
- Bias-Variance tradeoff

## Requirements

- Python 3.14+
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
- statsmodels

## Usage

Open the notebook with Jupyter:

```bash
jupyter notebook week1.ipynb
```

Or use VS Code with the Jupyter extension installed.

## Dataset Sources

Data downloaded from:
- ISLR R package
- MASS R package
